\chapter{CEIP-derived national subsector allocation weights}
\label{app:ceip-weights}

This appendix lists illustrative national shares (dimensionless, summing to one across
subsector columns within each pollutant row) used when blending spatial proxies after
reported-emissions (CEIP-style) processing. Table headers are shortened labels aligned
with the sector configuration (full YAML keys are documented in the proxy config files).
For waste (GNFR~J), weights partition reported emissions across solid, wastewater, and
residual families. For off-road transport (GNFR~I), shares refer to the three CEIP legs
(rail, pipeline, non-road) before fine-grid combination.
