Area weights debug maps — by sector
Source: proxy/visualizers/area_weights_map.py

Output modes (proxy.entry.MAP_TYPE):
  INTERACTIVE  — single Folium HTML, all layers togglable
  FIXED_IMAGE  — one PNG per layer (OpenStreetMap basemap + CAMS cell outlines)

Sectors WITHOUT area-weight maps:
  F_Roads      — not implemented
  H_Aviation   — point matching only

Dynamic layers: names in {braces} come from sector config / run data (inventory groups,
subgroups, pollutants, OSM slots, etc.). One map layer is emitted per key present at runtime.

================================================================================
A_PublicPower
================================================================================
Inputs / masks:
  - CORINE mask
  - Population z-score

Weights:
  - Weight W — {pollutant}          (one map per pollutant in the run)

B_Industry
================================================================================
Inputs / masks:
  - CORINE L3 121 industrial
  - CORINE L3 131 mineral
  - OSM refineries                  (osm_refineries_petroleum)
  - OSM manufacturing               (osm_manufacturing_combustion_residual)
  - OSM mineral                     (osm_mineral)
  - OSM chemical metal              (osm_chemical_metal)
  - OSM {group}                     (one per key in osm_by_group)

Weights:
  - W inventory group — {group}     (W_{group}, one per inventory group)
  - W fused pollutant — {pollutant}   (wp_{pollutant}, one per pollutant)
    e.g. W fused PM10, W fused CO
